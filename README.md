什么是 SchoolPower-iOS
-------
SchoolPower-iOS 是使用 Swift 编写的开源、现代化的 [PowerSchool](https://www.powerschool.com/) iOS 客户端替代品，原理是使用爬虫获取 PowerSchool 上的信息。
